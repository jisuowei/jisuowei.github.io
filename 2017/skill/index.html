设计 / Design

常言道：“少壮不努力，老大干设计。” 然而射鸡师也是要很努力的。

03年小学五年级一堂美术课上，老师用火苗燃烧的线条描出了一个“火”字，老师说这就叫设计。似乎就是从那一天起就特别想要做一些关于设计的事情。看着好多搞设计的人都有工作室和LOGO，我那幼小的心灵深受感染。

08年，在家用WindowsXP自带的画图工具给自己做了一个LOGO；两年后我又在刻字店里做成了印章；2014年用了点PS效果更新了LOGO；2015年到现在，因为没有手绘基础，数码风格，IMPACT字体跟手写字体都尝试了一下，都不是很满意。

最早在家里用那台没联网的台式机时，望着出神，发现“滚动条”其实是可以画出来的，就用画图工具捣鼓了一下午，根据在网吧里的记忆画出了一个音乐下载网页（当年的网页比较简单也比较丑），爸爸下班回来看见后，居然问是怎么联网的？ 哈哈哈，后来才知道，这TM不就是UI嘛！可惜没能保存下来。

高中用画图工具给住宿的同学做的假胸卡，去文印店打印的时候看老板用PS画布任意角度旋转，几个快捷键噼里啪啦就搞好了，很是羡慕；到了大二，终于有机会认真学起PS来，去图书馆借书看，网上找教程，做过的一些 壁纸 。

又开始学AE和FLASH，铺的太大了，直到参加工作以后，才又开始接触到AI、UI/UE、VI，独立完成过一个产品三个版本的流程、设计、切图。

前端 / Front-end

小时候我就想给自己做一个网站，鬼晓得现在做一个网站要学这么多东西。前端，入门易，深入难，框架多，更新快.. 若没有幼时梦想的支撑和持续的学习态度，前端，会很累。

身为半路出家自学派，当年看的视频教程，该教程计划是一周学完前端（现在想想都觉得好笑），1天HTML，2天CSS，4天JS。前三天感觉还可以，一学JS就懵逼，尼玛，什么鬼啊，深陷自我怀疑的泥潭。好在我没有放弃，硬着头皮学了下来，通过PS的学习过程已经让我明白，不懂，肯定是因为有基础出了问题；不知道函数怎么用，参数是干嘛的，更别提什么闭包、作用域.. 一听就头大。坚持着，学了四五位老师出的基础课程，依旧迷惑，直到自己尝试写一些自己用的代码，才渐入佳境。

逐渐理解掌握JS（ECMAScript、BOM、DOM）后，玩起了jQuery（源码敲过三遍），好用的一米，紧接着又学了AJAX，以及用jQuery的AJAX遍历JSON数据，感觉是越来越上手，有种打通了的感觉。

最近就是公司产品架构调整，做混合模式，正在尝试AngularJS中..

后台 / Back-end Server & DataBase

一年多时间学习了韩顺平老师两个月的Java，四个月的PHP教程，中间也穿插着一部分前端知识，以及别的老师关于服务端和数据库的讲解。

Java部分几乎停留在语法的讲解上，没有参与到项目中去，所以至于MVC具体是什么鬼并没有吃透；但也正是这段对于Java语法的学习，让我从侧面加强了对JavaScript的一些特点（比如什么叫泛型）的认识，更深层次的了解面向对象的封装继承多态三个特征。PHP的课程里又学了语法，对，语法几乎都一样；其中有一个小型项目，了解了数据传输的过程，页面如何通过超全局函数传值，GET、POST差异；本地服务器的搭建，以及MySql建库；分页处理；数据简单的增删改查；COOKIE和SESSION的操作，登录验证；虽然已经忘掉了一些具体实现方法，但对于前端知识的归纳总结起到了很大的帮助（插数据，字符串处理）。

最近正攻读《HTTP权威指南》，希望通过对底层数据传输过程的掌握来更好的做好前端工作。

机械 / Machine

如前所说，我本身学的是机械专业，选专业的时候我犹豫要不要选计算机，觉得高中数学没学好会影响到计算机的学习，最终选择了父亲的绘图道路。

双眼面对着晦涩难懂的机械书籍，身体应付着工厂疲惫不堪的劳动，内心承受着人生价值观的鞭挞。

然而就在看完小米2s发布会（一场号称看完就会成为米粉的发布会），看着雷军的成功之后，我重新燃起了学计算机的狂热。

虽然在机械上花费了一段宝贵的时间，但人生总避免不了走弯路，不把路都走一遍，你哪知道哪条路才是对的呢。钱家斌对我说：“猥琐鸡通过努力摆脱了工厂，改变了人生的轨迹。” 我说：“这本来就是我的人生啊。”
